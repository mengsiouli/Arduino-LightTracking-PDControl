# Arduino-LightTracking-PDControl

自動追蹤人體之廣告看板控制系統，基於 **Arduino MKR WiFi 1010**，整合編碼器回授與 PD 控制。

---

## 功能特色

- **光源追蹤**：透過光感測器偵測光源所對應之電壓值，並與環境光基準電壓比較，以調整廣告板之角度，使其朝向人體方向。
- **PD 馬達控制**：利用位置誤差與速度回授，精準控制伺服馬達轉動角度。
- **自動回原點功能**：當停在同個位置超過5秒或超出角度限制時，自動回到初始位置。

---

## 硬體需求

- Arduino MKR WiFi 1010  
- 伺服直流馬達
- 驅動電路
- 光感測器
---

## 腳位說明

| Arduino Pin | 功能                   |
|------------|------------------------|
| A0         | 控制馬達轉速            |
| A1         | 讀取光感測器數值        |
| 0          | 編碼器 A               |
| 1          | 編碼器 B               |
| 4          | 輔助光源控制           |

---

## 軟體說明

- 使用硬體計時器中斷（TC4）建立固定頻率的控制迴圈。
- 光感測器數值經過一階低通濾波，以提升控制穩定性。
- PD 控制器公式：controlSignal = -Kp * positionError + Kd * velocity。
- 超過設定角度範圍或卡住時，自動進入回零模式。

---

## 使用方法

1. 將程式上傳至 Arduino MKR WiFi 1010。
2. 連接硬體，如光感測器、帶有編碼器的馬達。
3. 打開序列埠觀察角度與光感測器數值。
4. 系統會自動根據反射光所造成的電壓差，調整板子角度，並防止過轉或卡住。
